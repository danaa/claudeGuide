    אתגר הבוס הסופי

    בחרו מוצר או רעיון שאתם אוהבים, תנו ל-Claude קונספט, והוא יעשה את כל השאר:

    1. פתחו פרויקט חדש : תיקייה חדשה + CLAUDE.md
    2. תנו קונספט : "אני רוצה פרסומת ל..." (משפט אחד מספיק!)
    3. /script : Claude חוקר ברשת וכותב תסריט מקורי
    4. /storyboard : Claude מייצר סטוריבורד מהתסריט
    5. /voiceover : Claude כותב ומקליט קריינות
    6. /animate : Claude עושה אנימציה עם תנועות מצלמה מתאימות
    7. /render : Claude מרכיב לווידאו סופי
