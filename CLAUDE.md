# Claude Code Quest

## About
Interactive Hebrew web game teaching non-technical creatives to use Claude Code.
Philosophy: **Claude does all the tech. You learn to talk to it.**

## Rules
- **NEVER use em dashes** (the long dash character). Use colons, commas, or periods instead.
- Always use the frontend-design skill for UI work
- Hebrew RTL interface
- Single HTML file, no build tools

## Game Flow (12 chapters)
0-Welcome, 1-Install, 2-FirstChat, 3-CLAUDE.md persona, 4-WebSearch,
5-WhatIsAPI, 6-Python+venv, 7-Storyboard/Gemini, 8-Voiceover/ElevenLabs,
9-Animation/xAI, 10-Remotion, 11-Skills, 12-FinalBoss

## Design
Uses Anthropic's official frontend-design skill.
Aesthetic: Warm editorial meets retro terminal.
Fonts: Outfit (display/body) + JetBrains Mono (code) + Rubik (headings).
Color: Dark warm stone palette with claude-orange accent.
